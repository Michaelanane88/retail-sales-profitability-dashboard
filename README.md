# Retail Sales & Profitability Dashboard

## Project Overview
This project analyzes retail sales and profitability data using SQL and Power BI to identify revenue trends, regional performance, and operational inefficiencies impacting profitability.

## Tools Used
- SQL
- Power BI
- Excel

## Key Business Insights
- West region generated the highest sales and profit.
- Technology was the most profitable product category.
- Several sub-categories generated negative profit despite strong sales activity.
- Discounting alone did not fully explain profitability issues.
- Consumer segment generated the largest share of sales revenue.

## Dashboard Features
- KPI tracking for Sales, Profit, Quantity, and Profit Margin
- Regional sales analysis
- Profitability analysis by category
- Loss-making sub-category investigation
- Customer segment sales breakdown

## Files Included
- Power BI dashboard (.pbix)

- Dashboard screenshot
- Dataset used for analysis
